Flask MongoDB API

Uma simples API de Filmes: desenvolvida com Flask e MongoDB

GET Request

curl -X GET http://localhost:5000/api

curl -X GET http://localhost:5000/filmes

curl -X GET http://localhost:5000/filmes/id

POST Request

curl -d '{"titulo":"Akira","generos":["Dystopian"],"ano_lancamento":1984}' -H "Content-Type: application/json" -X POST http://localhost:5000/filmes

PUT Request

curl -d '{"titulo":"Akira","generos":["Dystopian","Anime"],"ano_lancamento":1988}' -H "Content-Type: application/json" -X PUT http://localhost:5000/filmes/id

DELETE Request

curl -H "Content-Type: application/json" -X DELETE http://localhost:5000/filmes/id
